# Chess.JP

Chess.JP（表示名: Classical-chess.jp）は、ブラウザで遊べるチェスゲームです。

公開版: https://leoleoship.github.io/Classical-Chess.JP/

## 起動方法

公開版を使うか、このディレクトリでローカルサーバーを起動します。

```sh
python3 -m http.server 4173
```

その後、`http://localhost:4173` を開いてください。

`chess.js`、Supabaseクライアント、オンライン対戦に外部サービスを使用するため、インターネット接続が必要です。

## 主な機能

- 対人戦、Bot戦、オンライン対戦、チェスパズル、Board Maker、個人ルール
- `chess.js` による合法手、チェック、チェックメイト、キャスリング、アンパッサン、昇格の判定
- Beginner / Novice / Intermediate / Advanced / Grandmaster のBot難易度
- Advancedは相手の返しを、Grandmasterはさらに次の応手まで読む軽量ミニマックス探索
- Bot戦では白・黒を選択可能
- Bot戦ELO: 勝利 `+15`、敗北 `-10`
- チェスパズルは5段階の難易度に各5問、合計25問。メイトイン1からサクリファイスを含む複雑なメイトまで収録
- パズルでは相手の応手を自動再生し、難易度別の選択とクリア状況をブラウザに保存
- Supabase Realtimeを使ったオンライン人数表示、マッチング、指し手同期、投了
- オンライン対局は再読み込み後も保存済みルームへ再接続し、相手から最新局面を同期
- オンラインELO: 勝利 `+15`、敗北 `-10`
- Board Makerで任意の駒を配置し、盤面の保存、読込、削除、その局面からの対局が可能
- 個人ルール: キャスリングなし、アンパッサンなし、昇格はクイーンのみ、ルークとビショップは1マスだけ、ポーンの2マス移動なし、取れる場合は必ず取る、同じ駒を連続で動かせない
- 日本語 / 英語、Classic / Bright / Nightテーマ
- 効果音、音量調整、ブラウザ内で生成する落ち着いたBGM
- 盤面反転、再読込、投了、一手戻す、FENの表示・コピー・読込
- 棋譜、取られた駒、手数、手の評価を表示
- チェック、アンパッサン、昇格、キャスリング、戦術、チェックメイト、ドローなどの演出

## 手の評価

手の評価は軽量な局面解析による目安です。チェスエンジンの厳密な解析ではありません。

- `Brilliant`: 実際に駒を差し出すサクリファイスから、メイトや大きな戦術的利益を得る手。直近5回の自分の手にあるサクリファイスと、複数の駒による協力メイトも確認します。
- `Good`: 評価を保つ手、駒得、守られた有効手、成立している戦術など。
- `So-So`: 最善手との差が小さくないものの、決定的ではない手。
- `Bad`: 小さめの損失や、十分な見返りがないサクリファイス。
- `Mistake`: 大きな評価損、失敗した重大なサクリファイス、メイトイン1の見逃し、止められたメイト脅威の放置など。
- `Forced`: 合法手が1つだけの場合、または探索範囲内ですべての合法手が強制メイトにつながり回避不能な場合。

通常のクイーンによるチェックメイトだけでは `Brilliant` になりません。

## オンライン対戦について

オンライン対戦の通信にはSupabase Realtimeを使用しています。2つの端末または異なるブラウザで公開版を開き、それぞれ名前を保存して `Find match` を押すとマッチングできます。

現在のアカウントID、表示名、オンラインELOは各ブラウザの `localStorage` に保存されます。Supabase Authを使ったログインアカウントや、サーバー側で保護されたELOランキングではありません。ブラウザの保存データを消すと、それらもリセットされます。

オンラインルームと局面の再接続情報は最大8時間ローカル保存されます。両方のプレイヤーが切断している間の指し手を、サーバーのデータベースへ永続保存する仕組みは現在ありません。

## Variant Lab ロードマップ

次の案は、標準8×8盤を前提とする `chess.js` の合法手フィルターだけでは安全に実装できないため、専用の盤面・合法手エンジンを作る段階で追加予定です。

- Gravity Chess: 盤面を横向きにし、手の後に駒を重力方向へ落とす
- Long Board: 4×16盤と専用初期配置
- Powerful Pawns: ポーンの横移動と後退
- Place Captured Pieces: 取った駒を自分の色として盤上へ1回配置
